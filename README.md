# Oliyad Portfolio 2.0

A clean, modern, and fully responsive personal portfolio website designed to showcase my software engineering journey, technical skill set, and professional projects. Built using semantic HTML5, clean CSS3 layouts, and vanilla JavaScript.

👉 [Live Demo](https://olibbeto.github.io/Portfolio2.0/)

---

## 🚀 Features

- Interactive UI/UX: Features a fluid background using an interactive canvas particle effect (`particles-canvas`).
- Dynamic Theme Management: Includes a seamless Light/Dark mode toggle with persistence and smooth icon transitions.
- Multi-Page Architecture: Organized navigation layout across specific, dedicated pages:
  - index.html — Dynamic Landing Page & Home introduction
  - about.html — Developer background, profile, and overview
  - skills.html — Visual breakdown of core tech stack and proficiencies
  - experience.html — Professional timeline and technical background
  - portfolio.html — Project showcase and interactive deployment links
- Responsive Architecture: Pixel-perfect optimization across smartphones, tablets, and high-resolution desktop monitors.
- Modern Iconography: Integrated with FontAwesome 6.4.2 solid and custom icon sets.

---

## 🛠️ Tech Stack & Tools

- Frontend Core: HTML5, CSS3, JavaScript (ES6+)
- Visuals & Effects: HTML5 Canvas, Custom CSS Transitions
- Libraries & Assets: FontAwesome Icons, Google Fonts
- Hosting & Version Control: Git, GitHub Pages
